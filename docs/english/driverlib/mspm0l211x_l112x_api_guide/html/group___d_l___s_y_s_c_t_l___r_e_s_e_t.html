#define DL_SYSCTL_RESET_BOOT   (SYSCTL_RESETLEVEL_LEVEL_BOOT)

Perform a Boot reset.

This triggers execution of the device boot configuration routine and resets the majority of the core logic while also power cycling the SRAM
