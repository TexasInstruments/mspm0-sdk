§ period

uint32_t DL_Timer_PWMConfig::period

Specifies the pwm period. The actual value configured to LOAD register depends on the pwmMode selected. When pwmMode = DL_TIMER_PWM_MODE_EDGE_ALIGN, LOAD = (period -1). When pwmMode = DL_TIMER_PWM_MODE_CENTER_ALIGN, LOAD = (period / 2). Actual PWM freq = (Timer_freq/(prescaler * divideRatio * period))
