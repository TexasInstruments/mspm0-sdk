§ DL_DAC12_INTERRUPT_DMA_DONE

#define DL_DAC12_INTERRUPT_DMA_DONE   (DAC12_GEN_EVENT_IMASK_DMADONEIFG_SET)

Interrupt raised as an Acknowledgement to the DMA completing a transfer.

During DMA operation only, once the DMA has performed its desired number of transfers to the FIFO, the DMA will assert a DMA done event. The DMA will also send a status, which is 0 if there is more data to send. If there is a nonzero status, and the DMA is done sending information, the DMATrigger will be disabled and this interrupt will be asserted.
