__STATIC_INLINE void DL_GPAMP_init ( DL_GPAMP_Config *  config)

Initialize the GPAmp peripheral.

Initializes all the common configurable options for the GPAmp peripheral. Any other custom configuration can be done after calling this API. The GPAmp is not enabled in this API.
