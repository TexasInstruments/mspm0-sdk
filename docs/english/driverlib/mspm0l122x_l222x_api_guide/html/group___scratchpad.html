Detailed Description

Overview

The Scratchpad Memory Library allows full configuration of the MSPM0 Scratchpad Memory within the LFSS IP.

The LFSS scratch pad memory is a 16- to 128-byte register based memory, which will retain data similar to a non-volatile memory as long as VBAT is supplied. The number of implemented bytes shall be configurable in a granularity of 4 bytes. This means the scratch pad memory retains data when the main VDD supply is lost or during the SHUTDOWN mode. However, the memory content wonâ€™t be retained, when the VBAT domain itself is lost.
