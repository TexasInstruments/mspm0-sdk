Input/Output (TIO)

Overview

The tamper I/O is a GPIO which is sourced by the VBAT power domain. The LFSS IP will contain up to 16 I/O’s of this type. The I/O’s have two modes of operation. In IOMUX mode the tamper I/O input and output path are connected to the SoC IOMUX module and the I/O’s can be used as a 2nd function like SPI, UART or Timer PWM. The second mode is the Tamper mode. In this mode the I/O is completely controlled by the LFSS IP and will remain functional during the power loss of the main supply or during SHUTDOWN mode.
