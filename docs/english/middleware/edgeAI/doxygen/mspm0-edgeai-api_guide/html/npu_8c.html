void NPU_setInstrMem ( uint32_t  byteOffset,
uint32_t  num,
const uint32_t *  src 
)

Set the instruction memory of the NPU at the specified offset.

Parameters
[in]byteOffsetThe offset from the beginning of the NPU instruction memory to start the write from. This number MUST be a multiple of 4 bytes. If (byteOffset%4) != 0, a hard fault may occur.
[in]numThe number of 32-bit words to set.
[in]srcA pointer to the 64-bit aligned instructions in system memory to load to the NPU memory.
Returns
None
