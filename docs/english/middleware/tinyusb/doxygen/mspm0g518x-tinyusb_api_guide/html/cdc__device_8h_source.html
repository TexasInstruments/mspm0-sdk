26 /*!****************************************************************************
27  * @file cdc_device.h
28  * @brief CDC (Communication Device Class) Device Driver implementation
29  *
30  * @details This file implements the CDC device driver for TinyUSB stack.
31  * It handles the USB CDC protocol including:
32  * - Device enumeration and configuration
33  * - Serial data transfer between USB host and device
34  * - Control requests handling (line coding, control signals)
35  * - FIFO buffer management for TX and RX data
36  *
37  * The CDC implementation supports:
38  * - Multiple CDC interfaces
39  * - Configurable buffer sizes via CFG_TUD_CDC_RX_BUFSIZE and CFG_TUD_CDC_TX_BUFSIZE
40  * - Line coding configuration (baud rate, data bits, parity, stop bits)
41  * - Line state signaling (DTR/RTS)
42  * - Character matching for simplified input processing
43  *
44  * <hr>
45  ******************************************************************************/
