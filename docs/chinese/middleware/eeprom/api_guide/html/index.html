MSPM0 EEPROM 仿真库概述

引言

许多应用程序都需要将数据存储在非易失性存储器中，这样，即使在系统再次上电后 也可以重复使用或修改数据。EEPROM 非常适合此类应用程序。尽管 MSPM0 MCU 没有内部 EEPROM，但 MSPM0 的内部闪存支持 EEPROM 仿真。与使用 外部串行 EEPROM 相比，使用内部闪存的 EEPROM 仿真可节省引脚用量和 成本。

MSPM0 SDK 为 EEPROM 仿真提供两种解决方案：

  • 如果用户应用程序需要存储大块数据，那么 EEPROM 仿真 A 型解决方案很适合。
  • 如果用户应用程序需要存储小型可变数据，那么 EEPROM 仿真 B 型解决方案很适合。
