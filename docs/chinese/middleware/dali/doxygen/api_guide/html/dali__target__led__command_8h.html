void DALI_LED_enableDeviceType ( void  )

enables the control gear to execute LED specific commands

This function sets a flag indicating that the control gear can execute any of the LED specific functions per IEC 62386-207. This flag resides in the LED Control Gear variable that is stored within the overall control gear variable.
