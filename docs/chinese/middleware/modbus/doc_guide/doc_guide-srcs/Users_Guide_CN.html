MSPM0 Modbus库用户指南¶

1. 简介¶

MSPM0 Modbus库提供了一种简单的方法来开始使用MSPM0器件开发Modbus应用程序。

2. Modbus库功能¶

该库阵列是根据Modbus V1.2规范开发的。

3. Modbus示例用法¶
