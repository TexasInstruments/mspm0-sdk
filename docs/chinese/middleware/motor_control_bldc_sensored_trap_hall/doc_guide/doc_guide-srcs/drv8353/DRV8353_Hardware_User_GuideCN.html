DRV8353RHEVM Hardware User Guide¶

1. Overview¶

To spin the BLDC motor with the DRV8353RHEVM, the user needs to modify some jumpers on the MSPM0 LaunchPad, connect the EVM pins to the LaunchPad, connect the motor phases and hall sensors to the DRV hardware and power on the supply.

2. Hardware Required¶

  • LP-MSPM0G3507 - MSPM0G3507 Launchpad

  • DRV8353RHEVM - DRV8353RHEVM evaluation module

  • DC Power supply that supports the required voltage and current

  • BLDC Motor

  • Micro-USB cable

3. Hardware Setup¶

  1. Connect the jumpers J21 and J22 on the LP-MSPM0G3507 to the BP position

  1. If using External ADC reference resistor R22 on LP-MSPM0G3507 needs to be populated and your reference voltage should be applied to PA23 accessible on the bottom header row of the launchpad.

  2. Connect the LP-MSPM0G3507 to the DRV8353RHEVM in one of two ways:

    • Use jumper wires as shown in the table below. Refer to the schematic connector pinout for the locations of the DRV8353RHEVM signals.

    • Directly mate the Launchpad to the DRV8353RHEVM. Note, you will need to use male headers to mate the two rows of the Launchpad into the DRV8353RHEVM.

  3. Connect the 3 phases of the motor to OUTA, OUTB, and OUTC of the motor connector terminal block on the DRV8353RHEVM and the HALLA/B/C, Hall power, and Hall GND signals to the Hall connector block. Note: The user should identify the correct phases of motor from motor datasheets, this will vary from motor to motor.

  4. Connect the positive and negative of the power supply to VM and PGND/AGND of the power connector terminal block to the DRV8353RH-EVM. Don’t turn on the power supply output.

  5. Connect a Micro-USB cable from the LP-MSPM0G3507 to the PC.

  6. Set the power supply to a motor voltage within the DRV8353 specifications. Turn on the power supply. The green VM LED on the DRV8353RH-EVM should turn on.

  7. Once all the setup is done, spin the motor using the software or GUI. Refer to Software User Guide or GUI User Guide for more details.
