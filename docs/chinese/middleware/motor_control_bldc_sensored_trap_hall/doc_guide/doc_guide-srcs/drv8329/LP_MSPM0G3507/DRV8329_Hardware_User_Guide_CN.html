DRV8329AEVM Hardware User Guide¶

1. Overview¶

To spin the BLDC motor with DRV8329AEVM the user needs to modify some jumpers on the MSP0 Launchpad, connect the EVM pins to the LaunchPad, connect the motor phases and hall sensors to the DRV hardware and power on the supply.

2. Hardware Required¶

  • LP-MSPM0G3507 - MSPM0G3507 Launchpad

  • DRV8329AEVM - DRV8329AEVM evaluation module

  • DC Power supply that supports the required voltage and current

  • BLDC Motor

  • Jumper wires (to connect LP to EVM)

  • Micro-USB cable

3. Hardware Setup¶

  1. Connect the jumpers J21 and J22 on the LP-MSPM0G3507 to the BP position

  1. If using external ADC reference, populate resistor R22 on the LP-MSPM0G3507 and apply the external reference voltage to PA23, which is accessible on the bottom header row of the Launchpad.

  2. Connect the LP-MSPM0G3507 to the DRV8329AEVM using jumper wires as shown in the table below. Refer to the schematic connector pinout for the locations of the DRV8329AEVM signals.

  1. Connect the 3 phases of the motor to OUTA, OUTB, and OUTC of the motor connector terminal block on the DRV8329AEVM and the HALLA/B/C, Hall power, and Hall GND signals to the Hall connector block. Note: The user should identify the correct phases of motor from motor datasheets, this will varry from motor to motor.

  2. Connect the positive and negative of the power supply to VM and PGND/GND of the power connector terminal block to the DRV8329AEVM. Don’t turn on the power supply output.

  3. Connect a Micro-USB cable from the LP-MSPM0G3507 to the PC.

  4. Set the power supply to a motor voltage within the DRV8329 specifications. Turn on the power supply. The green VM LED on the DRV8329AEVM should turn on.

  5. Once all the setup is done, spin the motor using the software or GUI. Refer to Software User Guide or GUI User Guide for more details.
