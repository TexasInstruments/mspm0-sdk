§ DL_I2CT_STATUS_STALE_TX_FIFO

#define DL_I2CT_STATUS_STALE_TX_FIFO   (UNICOMMI2CT_SR_STALE_TXFIFO_MASK)

I2CT Stale TX FIFO.

Set = The TX FIFO data is stale. This occurs when the TX FIFO was not emptied during the previous I2CT transaction Clear = The TX FIFO data is not stale

§ DL_I2CT_STATUS_TX_MODE

#define DL_I2CT_STATUS_TX_MODE   (UNICOMMI2CT_SR_TXMODE_MASK)

I2CT TX Mode.

Set = is in the TX_DATA, TX_ACK, TX_WAIT, or ADDR_ACK state with the bus direction set to read Clear = is not in the TX_DATA, TX_ACK, TX_WAIT, or ADDR_ACK state with the bus direction set to read

§ DL_I2CT_STATUS_BUS_BUSY

#define DL_I2CT_STATUS_BUS_BUSY   (UNICOMMI2CT_SR_BUSBSY_MASK)

I2CT Bus Busy.

Set = The I2CT bus is busy. This is cleared on a timeout Clear = The I2CT bus is not busy. This bit is cleared when a STOP condition is detected on the bus.

§ DL_I2CT_STATUS_RX_MODE

#define DL_I2CT_STATUS_RX_MODE   (UNICOMMI2CT_SR_RXMODE_MASK)

I2CT RX Mode.

Set = is in the RX_DATA, RX_ACK, RX_WAIT, RX_ACK_WAIT or ADDR_ACK state with the bus direction set to write Clear = is not in the RX_DATA, RX_ACK, RX_WAIT, RX_ACK_WAIT or ADDR_ACK state with the bus direction set to write
