§ DL_LCD_BLINK_MEMORY_BIT_0

#define DL_LCD_BLINK_MEMORY_BIT_0   (LCD_LCDBM_MBIT0_MASK)

Bit 0 inside LCD blink memory index registers.

If LCD L[2*index] is selected as (1) segment line: 0b = LCD segment off 1b = LCD segment on (2) common line: 0b = Pin L[2*index] not used as COM0 1b = Pin L[2*index] is used as COM0
